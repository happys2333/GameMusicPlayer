# 🎮 GameMusicPlayer

这是一个专为游戏玩家和主播设计的 Python 桌面音效板（Soundboard）。
你可以通过**全局快捷键**在游戏内随时触发 Meme 音效（如“鸡你太美”、“退钱”、”新三国“等），并可以配合软件将音频输出到麦克风，与队友共享欢乐，节目效果拉满！

## ✨ 核心功能

* **⌨️ 全局快捷键捕获**：即使在游戏全屏状态下，按下快捷键也能秒切音效，无需切出游戏画面。
* **🔀 多音效随机播放**：支持为一个快捷键绑定多个音频文件。每次按下时随机抽取一首播放，保持新鲜感。
* **🎛️ 独立音频输出选择**：内置设备选择器，可以直接将音效输出到指定的虚拟声卡，不干扰你个人的游戏背景音。
* **🔊 全局音量控制**：内置平滑音量滑块，随时控制 Meme 音量大小，防止“炸麦”。
* **💾 自动记忆配置**：自动保存你的快捷键绑定、音量大小和输出设备，下次打开即用。
* **⏹️ 一键打断**：音效播放时，再次按下任意绑定的快捷键即可立刻停止播放。

## 🛠️ 环境依赖与安装

1. **安装 Python**：建议使用 Python 3.11 或 3.12 稳定版。
2. **克隆/下载本项目**到本地。
3. **安装必要的 Python 库**：
   打开终端并运行以下命令：
   ```bash
   pip install pygame pynput
   ```
